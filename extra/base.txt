عالی! پس بیایم یه مدل کمی دقیق‌تر بسازیم که **هر فاکتور یه وزن درصدی داشته باشه** و مجموعش تخمین عمر بده. من یه نسخه پایه طراحی می‌کنم که بشه راحت توی Python یا Excel محاسبه کرد.

---

## 1️⃣ پایه فرمول

فرض می‌کنیم:

[
\text{Estimated Age} = \text{Base Age} + \sum (\text{Factor Effect} \times \text{Weight})
]

* **Base Age**: عمر متوسط کشور (مثلاً 75 سال)
* **Factor Effect**: مقدار مثبت یا منفی هر عامل (مثلاً ورزش +3 سال، سیگار -10 سال)
* **Weight**: درصد تاثیر هر عامل (جمع وزن‌ها = 1 یا 100٪)

---

## 2️⃣ فاکتورهای نمونه و وزن‌ها

| فاکتور                   | اثر (سال) | وزن  |
| ------------------------ | --------- | ---- |
| ورزش                     | +3 تا +5  | 0.2  |
| سیگار                    | -10 تا 0  | 0.25 |
| BMI                      | -2 تا +2  | 0.15 |
| رژیم غذایی               | -2 تا +2  | 0.15 |
| الکل                     | -3 تا 0   | 0.1  |
| فشار خون / قند / کلسترول | -5 تا +3  | 0.15 |

> وزن‌ها درصدی از تاثیر کل هستن. می‌تونیم بعداً بر اساس تحقیقات دقیق‌تر تغییرشون بدیم.

---

## 3️⃣ نحوه امتیازدهی فاکتورها

**ورزش (Exercise)**

* ≥ 150 دقیقه/هفته → +5
* 75-150 دقیقه → +3
* 0-75 دقیقه → 0

**سیگار (Smoking)**

* روزانه ≥ 1 پاکت → -10
* گاه‌گاهی → -5
* نمی‌کشد → 0

**BMI**

* 18.5–24.9 → +2
* 25–29.9 → 0
* ≥ 30 یا <18.5 → -2

**رژیم غذایی (Diet)**

* سالم (میوه، سبزی، پروتئین کافی) → +2
* متوسط → 0
* ناسالم → -2

**الکل (Alcohol)**

* مصرف زیاد → -3
* مصرف کم → 0

**فشار خون / قند / کلسترول (Health Metrics)**

* همه نرمال → +3
* یکی غیر نرمال → 0
* چند فاکتور غیر نرمال → -5

---

## 4️⃣ فرمول نهایی (مثال)

[
\text{Estimated Age} = BaseAge + 0.2 \times ExerciseScore + 0.25 \times SmokingScore + 0.15 \times BMIScore + 0.15 \times DietScore + 0.1 \times AlcoholScore + 0.15 \times HealthScore
]
